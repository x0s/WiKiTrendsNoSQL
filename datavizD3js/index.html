Bienvenue

Le but du projet est de reproduire un dashboard resemblant au site trendingstopics.org qui presente, pour une journee donnee, quels ont ete les sujets les plus vus sur Wikipedia.

On s’interesse principalement aux Trending Topics - les sujets les plus vues sur les 30 derniers jours et les Raising Topics - les sujets les plus vus dans les 24 heures.
